N Scale Train Configurator

Open index.html in a modern browser. No web server or build process is required.

Features:
- Select between 1 and 8 cars
- Randomized draw without duplicate cards in the same consist
- Reshuffle button
- Printable selected consist
- Expandable catalogue of all eight cards

Files:
- index.html
- styles.css
- app.js
- cars.js
- assets/*.jpg

To add another car:
1. Add its image to assets/.
2. Add another object to the CARS array in cars.js.
3. Increase the max value on the number input in index.html if the limit should exceed 8.
